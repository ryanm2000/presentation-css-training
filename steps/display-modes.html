Block
  • By default full width
  • Will start and end with a new line
  • Can be padded, margined as much as you like
  • The default of P, Div, Section, blockquote
Inline
  • Will each sit next to each other
  • Padding doesnt work as expected
  • Margin only works horizontally
  • The default of strong, anchor, span
Inline Block
  • A mixture of the 2
  • A good alternative to float (discussing in 2 slides)
