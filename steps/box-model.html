Box Model

box-sizing: content-box; // Default
box-sizing: padding-box; // Width + padding
box-sizing: border-box;  // Width + padding + border

Further Reading: About the Box Model on csstricks.com

  • Every element on the web is a rectangular box
  • Content-Box : real width = width + padding + border
  • Border box allows you to set a width and not worry about padding screwing with it
